TethysChloris.jl

TethysChloris.jl is a Julia implementation of the Tethys-Chloris ecohydrological model for plot scale applications. Tethys-Chloris is a mechanistic model designed to simulate essential components of the hydrologic and carbon cycles, resolving exchanges of energy, water, and CO2 between the land surface and the planetary boundary layer with an hourly time step.

The original Tethys-Chloris model was developed in MATLAB by Prof. Simone Fatichi.
